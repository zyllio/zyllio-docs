Publications

You can publish your application at any time, even if your application is not completely finished and fully functional.

Publish Your Application

The publishing process is accessible from the main toolbar. It allows you to quickly publish a mobile application and make it immediately available to users.

The application can be downloaded onto a mobile phone via the displayed QR code or opened in desktop mode on a PC or Mac. Additionally, the application's URL can be copied for easier sharing.

Sharing Screen

Download via QR Code

Once the application is published, a unique QR code is generated. Users can scan this code with their mobile phone's camera to download and install the application directly onto their device.

This method is quick and does not require access to an app store (such as Google Play or the App Store) to complete the installation.

Desktop Mode Access

The application can also be opened and used in desktop mode on a PC or Mac. This allows for testing or browsing the app on a larger screen.

The "Open desktop" button displayed on the screen allows you to directly open the application in a compatible browser on a computer.

URL Sharing

A "Copy link" button allows you to copy the application's URL to the clipboard. This makes it easy to share the link with other users, whether by email, message, or social media.

The link will redirect users to the application, either on their mobile device or computer, depending on the device they are using.

Using a Custom Domain

If the application has a custom domain configured, the generated URL for sharing will use the custom domain instead of a default generic URL.

This feature is useful for branding the application with a trusted domain, providing better recognition and credibility

To use a custom domain, ensure that the domain is properly configured in your application's settings. This guarantees that the shared URL will use the custom domain instead of the default system-generated URL.
