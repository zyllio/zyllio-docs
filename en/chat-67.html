Chat

Add in-app conversations to improve support, collaboration, and user engagement.

Chat experience design

  • Define conversation context per user or business object.
  • Display read status and message chronology clearly.
  • Plan moderation or escalation rules for operational teams.

Implementation steps

  1. Model message entities and participant metadata.
  2. Bind conversation lists and detail views in Screen editor.
  3. Use Variables for active thread and draft state.
  4. Validate real-time behavior and offline fallback in Simulation.

Operational note

Track response time KPIs and message delivery failures to keep chat quality high in production.
