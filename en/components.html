Zyllio Studio Components Hints

This document provides a comprehensive list of all Zyllio components with their descriptions (hints).

Basics

Label Description
Bulleted list A list component that displays items as a bulleted or numbered sequence. It supports icon customization and custom list formatting options. Perfect for displaying features, benefits, or sequential steps.
Header A top navigation header bar that displays a page title along with optional actions like back buttons or custom action buttons. It supports custom alignment, gradients, borders, and typography. Used to provide structure and navigation consistency at the top of screens.
Icon A component that displays a single customizable icon from standard icon packs alongside optional text. It supports custom sizes, colors, background fills, and borders to serve as visual indicators or interactive buttons.
Image A versatile component designed to display images with options for different layouts, profile mode (circular cropping), labels, and descriptions. It supports click actions, custom borders, padding, and alignments. Essential for displaying visual content, user avatars, or media banners.
Label button A contact button that displays an icon, text, and an action trigger. It is configured to initiate direct interactions such as dialing a phone number, sending an email, or calling a custom action.
Label Text A dual-text component that displays a descriptive label alongside its main value. It supports horizontal or vertical layout orientations and full text styling options. Ideal for displaying key-value pairs, metadata, or form field summaries.
Markdown A rich text display component that parses and renders standard Markdown syntax. It allows the integration of headers, lists, links, and formatted text directly into the page layout. Ideal for showing formatted articles, policies, or dynamic documentation.
Navigation A navigation bar anchored to the bottom of the screen that allows users to switch between different screens. It supports displaying icons, text labels, badges, and custom actions. Ideal for building standard mobile application tab bars.
Options A component that displays multiple options in a horizontal or vertical list configuration. It supports custom colors, borders, and labels, making it ideal for displaying tags, categories, or filters.
Separator A simple spacing component used to create visual separation and distance between elements on a screen. It can display a solid line or act as empty spacing with a customizable height. Ideal for structuring layouts and improving readability.
Suggestion A box component designed to highlight tips, info alerts, or recommended actions with a leading icon and a text block. It supports custom icons, sizes, border radii, and background fills. Useful for guiding users or displaying contextual call-to-actions.
Text The foundational text component used to display labels, paragraphs, or copy with comprehensive styling options. It offers extensive control over typography including alignments, font sizes, weights, styles, colors, and gradients.

Buttons

Label Description
Button bar A horizontal group of multiple buttons. Used to present a set of related actions aligned next to each other.
Favorite A toggle button for favoriting items in a list or database. Automatically updates the relation between the user and the item.
Float A floating action button that hovers on top of the screen contents. Often placed in the bottom right corner for primary actions.
Follow A toggle button to follow or unfollow items, creators, or topics. Manages follow states and triggers customizable transition actions.
Icon A small circular button displaying only an icon. Designed for compact actions or icon-only triggers.
Like A toggle button to like or unlike content. Connects directly to a database table to keep track of user likes.
Link A button styled as a simple text hyperlink. Ideal for secondary actions, terms of service, or clean layouts.
Normal A standard solid action button. Designed for primary call-to-actions with customizable colors, labels, and click events.
Outline A button with a transparent background and a colored border outline. Commonly used for secondary or less prominent actions.

Contacts

Label Description
Download file A button configured to download a specified file. Useful for sharing PDF documents, assets, or resources.
Email A contact button that launches the default email client with a pre-filled recipient address. Ideal for direct support queries.
Export data A component that allows users to export data from tables or databases. Supports format configurations for downloading files.
Facebook page A button that redirects users to a specified Facebook profile or page. Helps increase social media presence.
Instagram A button that opens a specified Instagram profile. Perfect for linking to visual portfolios or social accounts.
Link A button that opens an external website link in a new browser tab. Perfect for linking to resources or external pages.
Phone A button that opens the device dialer with a pre-filled phone number. Allows users to start a phone call instantly.
Profile A profile information block displaying a user avatar, username, and metadata details. Ideal for profile cards or user directories.
QR Code VCard A component that generates and displays a QR Code containing contact details. Users can scan it to instantly save contact information to their phone book.
Twitter A button that links directly to a Twitter profile or page. Useful for sharing updates and connecting on social media.
VCard A contact card component that allows users to download a vCard (.vcf) file. Helps save contact details directly to the device contacts.
Whatsapp A contact button that opens a WhatsApp chat window with a specified phone number. Ideal for instant chat interactions.
Youtube channel A button that opens a specific YouTube channel or video link. Helps users discover video content.

Container

Label Description
Container A layout container component used to group and arrange child elements. It supports flex configurations, backgrounds, padding, and borders.

Dashboard

Label Description
Chart A visual data representation component supporting line, bar, pie, and doughnut charts. Connects directly to database tables to visualize values dynamically.
Progress A visual progress bar that displays completion percentages or ratios. Ideal for tracking tasks, goals, or loading states.
Statistic A dashboard card designed to highlight key metrics, counts, or summary values. Supports big labels and descriptive metadata text.

Form-fields

Label Description
Capture audio An audio recorder component allowing users to capture sound directly from their device microphone. Useful for voice notes.
Checkbox A standard checkbox input field for binary (yes/no) choices. Supports custom labels, default values, and change events.
Choose options An input component displaying options as selectable chips or pills. Allows users to pick single or multiple options easily.
Comment A text area input field designed specifically for writing comments. Includes placeholder options and action triggers.
Date A date input field with a built-in calendar picker. Helps users select dates accurately with custom format configurations.
Dropdown A selection dropdown menu input field. Ideal for selecting a single item from a long list of options.
Email An email input field optimized for email addresses. Includes basic syntax validation and virtual keyboard type support.
File A file uploader component allowing users to select and upload documents or assets. Supports file size limits and format constraints.
Image An image uploader component supporting drag-and-drop or device camera capture. Ideal for profile pictures or attachment uploads.
Message A text input field optimized for short messaging or chat messaging. Includes dynamic expansion and send action triggers.
Number A numeric input field restricted to number inputs. Supports minimum, maximum, and step parameters.
Password A secure text input field that masks characters. Useful for password entries, logins, and signup forms.
Phone number A phone number input field. Formats input data and triggers appropriate virtual keyboards on mobile devices.
Quantity A quantity picker input field with plus and minus buttons. Simplifies incrementing and decrementing numeric counts.
Radio A group of radio buttons for mutually exclusive options. Allows users to select a single choice from a visible set of options.
Rating A star-based rating component. Allows users to submit scores or feedback using interactive star ratings.
Signature A canvas-based signature field allowing users to draw their signatures. Ideal for approval workflows and forms.
Slider An interactive range slider input field. Allows selecting numeric values by sliding a thumb between minimum and maximum bounds.
Switch A toggle switch input field representing a boolean state. A modern alternative to checkbox inputs for settings.
Text A generic single-line text input field. The most common input type for names, titles, and general text entries.
Time A time input field supporting hour and minute selections. Helps users schedule events or specify times accurately.

Geolocation

Label Description
Map An interactive map component that displays geographical data and user locations. Supports custom layers and zoom configurations.
Marker A map pin/marker component used to pinpoint specific locations on a map layout. Supports popup descriptions and click actions.

Lists

Label Description
Calendar A calendar grid component that displays events, tasks, or schedules. Supports daily, weekly, or monthly layouts linked to data sources.
Card A content card list item displaying titles, descriptions, and media. Perfect for list items, news previews, or product listings.
Carousel A horizontal sliding container displaying a list of elements sequentially. Supports auto-play, navigation arrows, and paginators.
Categories A category navigation list displaying items as circular icons or tags. Useful for filtering content or navigating catalog sections.
Chat A dynamic chat thread component that displays messages with distinct sender and receiver styles. Connects to database tables for real-time messages.
Comments A discussion list component designed to display user comments. Supports avatars, timestamps, and replying triggers.
Gallery An image gallery layout component displaying photos in grid grids. Ideal for visual portfolios, albums, or media libraries.
Grid A flexible grid list layout component displaying database items in columns and rows. Highly customizable layout style.
List A standard vertical list layout component designed to display database items sequentially. Supports search, sort, and filter mappings.
Masonry An uneven masonry grid layout component. Perfect for Pinterest-style content grids with varying item heights.
Post A rich social post layout component displaying author profile details, text copy, images, and quick action buttons.

Media

Label Description
Audio An audio player component that plays audio tracks or streaming content. Supports volume, progress controls, and play/pause buttons.
Embed An iframe embedding component used to display external web content directly inside a page. Ideal for external forms or calculators.
QR Code A component that dynamically generates and displays QR codes from text, URLs, or variable values. Perfect for sharing links.
Video A native video player component that streams or plays video files. Supports standard media playback controls.
Youtube video A component designed to embed YouTube videos cleanly. Supports standard playback parameters and player configurations.

Payment

Label Description
Cart A checkout cart component showing items, quantities, and price summaries. Essential for online shopping and order submission workflows.

Tables

Label Description
Table A standard data table component used to organize and display relational data in columns and rows. Supports column styling and sorting.
Table editor A tabular data grid editor allowing users to directly insert, update, or delete rows. Ideal for backoffice management.

Website

Label Description
Banner A hero banner header component for website pages. Supports background images, titles, action buttons, and promotional messages.
