Formulas

Use formulas to compute values, validate inputs, and drive conditional behavior in your app.

Formula use cases

  • Derive display values from multiple fields.
  • Build conditional labels and visual states.
  • Validate user input before actions run.

Implementation pattern

  1. Keep formulas short and focused on one responsibility.
  2. Name computed fields explicitly for maintainability.
  3. Combine with Variables for state-dependent behavior.
  4. Validate outputs with real data in Simulation.

Quality tips

Avoid deeply nested expressions. Split complex logic into reusable intermediate fields when possible.
