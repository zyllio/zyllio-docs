Application settings

You can pass information to your Zyllio application directly through its URL. This makes it possible to control the display or pass data, such as a user name or an ID, as soon as the app opens.

URL structure

Parameters are added after your application ID, starting with a question mark (?).

Case A: Specify a specific screen (Optional)

If you want the application to open on a specific screen rather than the default home screen, place the screen name in the first position.

  • Example: https://zyllio.link/apps/<appId>/?SalesScreen

Case B: Pass data without changing the screen

If you only want to pass data without changing the start screen, list your parameters directly.

  • Example: https://zyllio.link/apps/<appId>/?Name=Didier

Combining multiple parameters

To combine the start screen and extra data, use the & symbol as a separator.

Golden rule: If you specify a start screen, it must be the first parameter after the ?.

Full example:

https://zyllio.link/apps/d0f269524cd4/?Dashboard&Name=Didier&Status=Premium

In this example:

  1. The application opens on the Dashboard screen.
  2. It passes the Name (Didier) and Status (Premium) variables.

How do you retrieve these values?

To use this data in your no-code logic, for example to display a welcome message or filter a list:

  1. Go to the Application section in the editor.
  2. Choose the "Application parameter" function.
  3. Enter the name of the key used in the URL (for example: Name).

Handling special characters (encoding)

URLs cannot contain spaces or accented characters directly. If your screen names or parameter values include them, they must be encoded.

Basic rules

  • Spaces: Replace them with %20.
  • Accents: It is strongly recommended to avoid them in your screen names. If you need to pass a value with an accent (for example, "Sebastien"), the URL will often transform it automatically.

Conversion examples

Desired text URL format
Dashboard screen ?Dashboard%20screen
First name = Heloise ?FirstName=H%C3%A9lo%C3%AFse
