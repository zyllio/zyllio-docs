Dashboard

Use the Dashboard as your command center to monitor projects, usage, and release readiness.

What you can do from Dashboard

  • Track your latest projects and jump back into editing quickly.
  • Review pending tasks before publishing an update.
  • Access shortcuts to design, simulation, and publication workflows.

Daily workflow recommendation

  1. Open Dashboard and review project status indicators.
  2. Continue implementation in Design tool and Screen editor.
  3. Run a quick end-to-end check in Simulation.
  4. Publish only when key checks are green in your release process.

Best practices

  • Keep naming conventions consistent across projects and screens.
  • Review errors and warnings before each publication.
  • Use plugins only when they bring clear value to your app scope.
