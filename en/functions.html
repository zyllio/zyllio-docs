Zyllio Studio Functions Hints

This document provides a comprehensive list of all Zyllio functions with their descriptions (hints).

Condition

Label Description
If then A simple conditional block that returns a specified value only if the defined condition is met.
If then else An conditional logic block that returns one value if a condition is true, and another if it is false.

Data

Label Description
Average in rows Calculates the average numeric value of a specified column across a set of table rows.
Count rows Returns the total number of rows in a specified list or table selection.
Count unique in rows Counts the number of distinct, unique values in a specified column across table rows.
Maximum in rows Finds the highest numeric value in a specified column across a set of rows.
Maximum text in rows Finds the maximum text value (alphabetically last) in a specified column across table rows.
Minimum in rows Finds the lowest numeric value in a specified column across a set of rows.
Minimum text in rows Finds the minimum text value (alphabetically first) in a specified column across table rows.
Rows to text Concatenates and formats values from a specified column across multiple rows into a single text block.
Search rows Filters and searches table rows matching specified query criteria.
Sum in rows Calculates the sum total of all numeric values in a specified column across a set of rows.

Date

Label Description
Add Adds a specified amount of time (days, hours, minutes) to a base date.
Current date & time Returns the current date and time of the system or device.
Date & time Constructs or formats a custom date and time value from specified inputs.
Duration from date range Calculates the time duration or difference between two specified date values.
Duration from today Calculates the duration or time elapsed between a specified date and the current day.
Substract Subtracts a specified amount of time (days, hours, minutes) from a base date.

Geolocation

Label Description
Distance Calculates the straight-line distance between two specified geographical coordinates.
Distance from position Calculates the geographical distance between a set coordinate and the user's current position.

Image

Label Description
Image Processes or resolves image source URLs and options for rendering.

Json

Label Description
JSON Query Queries and extracts specific values from a structured JSON payload using a search expression.

List-item

Label Description
Column Retrieves the value of a specific column from a list item or database row.
List item index Returns the 0-based index position of an item within a list or selection.
List item position Returns the 1-based display position of an item within its containing list.

Number

Label Description
Amount Formats a numeric value as a currency string with specified symbol and decimal settings.
Calculation Evaluates a mathematical expression combining numbers, variables, and standard operators.
Number Converts or formats inputs into a standardized numeric format.
Random number Generates a pseudo-random numeric value within a specified minimum and maximum range.

Payment

Label Description
Cart amount Calculates the total monetary amount of all items currently in the shopping cart.
Cart product count Returns the total number of unique products currently added to the shopping cart.
Cart quantity Returns the total aggregate quantity of all items in the shopping cart.
Print cart Generates a formatted text representation of the current shopping cart contents for printing or receipts.

Saas

Label Description
Application Id Retrieves the unique identifier of the running application instance.
Application parameter Retrieves the value of a specific global application configuration parameter.
Application Template Id Retrieves the template identifier associated with the running application.
Table Id Retrieves the unique database identifier for a specified table name.

String

Label Description
Capitalize Converts the first letter of the text input to uppercase and the rest to lowercase.
Concatenate Joins multiple text segments or variable values together into a single continuous string.
Include Checks if a specified search term exists within a target text string.
Join Concatenates elements of a list into a single string using a specified separator character.
Length Returns the number of characters in a specified text string.
Lower case Converts all characters in the text input to lowercase.
Matche expression Checks if a text string matches a specified regular expression pattern.
Occurences Counts how many times a search term appears within a target text string.
Repeat text Repeats a specified text string a set number of times.
Replace Replaces occurrences of a specified search string with a replacement string.
Replace expression Replaces substrings matching a regular expression pattern with a replacement string.
Split Splits a text string into an array of substrings based on a specified separator character.
Substring Extracts a portion of a text string starting at a specified index with a set length.
Text Converts a value or variable into a standardized text representation.
Unique Id Generates a unique random identifier (UUID/GUID) string.
Upper case Converts all characters in the text input to uppercase.
Validate IBAN Validates if a text string is a valid International Bank Account Number (IBAN).

Terminal

Label Description
Terminal Id Retrieves the unique identifier of the user's terminal or device.
Terminal language Retrieves the primary language setting of the user's terminal or browser.
Terminal OS Retrieves the operating system name of the user's terminal or device.
Terminal type Retrieves the device category or type (e.g. desktop, mobile, tablet) of the terminal.

Translation

Label Description
Translate Translates a key or label into the current active user interface language.
