TimeTonic

Zyllio lets you easily connect your TimeTonic databases to an application without needing programming knowledge.

Thanks to this integration, you can access, view, and interact with your TimeTonic data in real time, directly from your application.

Configuration

TimeTonic setup

Before you start importing your TimeTonic tables and views, you must configure the TimeTonic integration

Here are the steps to follow

  1. Sign in to your TimeTonic account
  2. Open your account settings
  1. Then select the Settings option in the left menu
  2. Click the Add a new API key button and select the options below
  1. Validate the API key creation
  2. Once created, click Copy to copy the API key
  1. Also note your account identifier from Account Information

Zyllio setup

  1. Sign in to Zyllio
  2. Select the application to integrate with TimeTonic
  3. Select the Settings tab, then TimeTonic
  4. Enable the TimeTonic integration
  5. Paste the API key copied from the TimeTonic portal and the user identifier as follows

Importing TimeTonic tables and views

From the Database tab, click the TimeTonic option in the Import table section on the left

Then select the table or tables to import. This operation can be repeated later to import other tables

Once validated, Zyllio imports all selected tables as well as field types and their relationships. Example of 5 imported tables with relationships

Calculated fields such as formulas, lookup, rollup, and id ... are marked with a lock. These fields cannot be edited from Zyllio

Adjustments are sometimes needed to give Zyllio more information about the field type. For example, the Image field in the Products table is a Link type because TimeTonic defines it as Files. To indicate that this attachment is an image, you only need to change its type from Zyllio, and Zyllio will automatically select this Image field in its components

Structure synchronization

When you make structural changes to your TimeTonic tables, for example adding a field, these changes do not appear in Zyllio

To synchronize these changes, click the Refresh columns button. Once synchronization starts, Zyllio will add and remove fields based on the changes

Zyllio never modifies the structure of TimeTonic tables in your TimeTonic account. Only the Zyllio definition can be modified from Zyllio

Data synchronization

Zyllio uses a data cache to improve access performance to TimeTonic based on how the TimeTonic tables are used. As a result, changes from TimeTonic or an external automation may not be visible immediately in Zyllio. A delay of up to 1 minute may be necessary

To force a refresh, click the Refresh data button
