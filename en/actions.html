Zyllio Studio Actions Hints

This document provides a comprehensive list of all Zyllio actions with their descriptions (hints).

Account

Label Description
Create application Creates a new application instance within the account workspace.
Publish application Publishes the current application version, making it live and accessible to users.
Sync application Synchronizes application state and configurations across workspace environments.
Update application Updates application metadata, settings, and workspace preferences.

Authentication

Label Description
Change password Triggers a password change request for the currently logged-in user.
Disconnect Logs out the current user, clearing session tokens and navigating to the sign-in screen.
Login Authenticates a user with a username and password, starting a secure session.
Reset password Sends a password reset email or code to help recovery for a user account.
Show user profile Displays the user profile management panel to edit personal details and settings.
Sign Up Registers a new user account with specified profile fields and credentials.
SSO Disconnect Logs out a user from their Single Sign-On (SSO) session.
SSO Login Initiates a Single Sign-On (SSO) login flow using configured identity providers.

Bluetooth

Label Description
Bluetooth connect Prompts the user to select and connect to a nearby Bluetooth Low Energy (BLE) device.
Bluetooth notification Subscribes to notifications or indications from a connected Bluetooth device characteristic.
Bluetooth write Writes value data bytes to a specific service characteristic on a connected Bluetooth device.

Calendar

Label Description
Install event Installs and adds a custom event with details directly to the device calendar.

Controls

Label Description
Continue if Halts workflow execution if a specified condition is not validated.
Début boucle Starts a loop sequence to execute nested actions repeatedly over a list of items.
Decision point Evaluates a condition during action execution to branch the workflow path.
Fin boucle Marks the end boundary of a loop block in the workflow sequence.

Data

Label Description
Create a row Inserts a new record row into a database table with configured column values.
Delete a row Deletes a specific row record from a database table matching a target identifier.
Delete rows Deletes multiple row records from a database table matching query filters.
Delete selected row Deletes the currently selected row in a list or table component.
Export CSV Generates and triggers a download of a CSV file containing records from a table.
Update a row Updates column values for a specific record row in a database table.
Update selected row Updates column values for the currently selected row in a list or table component.

Device

Label Description
Get current position Requests device GPS sensors to fetch the user's current geographic coordinates.
Voice recognition Starts voice recording and triggers speech-to-text recognition on the device.

Dialog

Label Description
Show confirmation Displays a confirmation dialog requesting the user to approve or cancel an action.
Show message Displays a custom modal alert message box with customizable text and close buttons.
Show notification Displays a temporary notification banner alert on the screen.

Messaging

Label Description
Send Email Sends an email message via Gmail with specified recipient, subject, and body.
Send SMS Sends an SMS text message using Twilio gateway integrations.
Send Whatsapp Sends a WhatsApp chat message using Twilio API configurations.

Navigation

Label Description
Close popup Closes the currently active popup screen window.
Go back Navigates back to the previous screen in the navigation history.
Go home Navigates directly to the designated home screen of the application.
Go to screen Navigates directly to a target screen specified by name or identifier.
Navigate website Redirects the browser or app to an external URL or web address.
Open popup Opens a designated screen as a modal popup window.
Refresh screen Reloads data and updates all components on the current active screen.
Wait Pauses workflow action execution for a specified duration in seconds.

Offline

Label Description
Offline sync Triggers synchronization of offline local database changes with the remote server.

Openai

Label Description
Generate Image Uses OpenAI Dall-E model services to generate custom images based on text prompts.
Generate Text Uses OpenAI GPT models to generate text responses from input prompts.
Speech to text Transcribes audio voice recordings into text logs using OpenAI Whisper model.
Text to speech Synthesizes spoken audio files from text copy using OpenAI voice models.

Payment

Label Description
Add product Adds a product item and quantity to the active shopping cart selection.
Cancel payment Cancels an ongoing subscription checkout payment flow.
Confirm payment Confirms and finalizes a pending payment transaction.
Create account Registers a user profile account with payment service providers.
Customer portal Redirects users to the payment billing management portal.
Process payment Initiates a checkout transaction flow for cart items.
Process subscription Initiates a recurring billing subscription flow for a checkout plan.
Remove product Removes product items or decreases their quantities in the shopping cart.
Reset cart Clears all product items and resets the active shopping cart to empty.

Push-notification

Label Description
Send to all Dispatches a push notification broadcast to all registered application users.
Send to channel Dispatches a push notification message to users subscribed to a target channel.
Send to user Dispatches a push notification message target to a specific user ID.
Subscribe Subscribes the device terminal to push notification alerts.

Qrcode

Label Description
Scan QR Code Opens the device camera interface to scan and decode a QR Code.

Rest

Label Description
Delete Dispatches a DELETE request call to a specified REST API endpoint.
Get Dispatches a GET request call to fetch resource payloads from a REST API endpoint.
Patch Dispatches a PATCH request call to modify specific resource fields in a REST API endpoint.
Post Dispatches a POST request call to submit payload data to a REST API endpoint.
Put Dispatches a PUT request call to write or replace resource data in a REST API endpoint.

Theme

Label Description
Zoom in text Increases the global theme typography size scale for text zoom in.
Zoom out text Decreases the global theme typography size scale for text zoom out.

Variable

Label Description
Clear selection Resets the currently selected row or item variable pointer to empty.
Decrement Decreases the numeric value of a specified variable by 1.
Increment Increases the numeric value of a specified variable by 1.
Load variables Loads and initializes variables into the application session state.
Reset variables Resets all session variables or a target variable back to their default values.
Set selection variable Sets a selection variable pointer to reference a target row item.
Set variable Assigns a specified static or dynamic value to a target variable.
Set variable if Assigns a value to a variable only if a specified condition is validated.

Webhook

Label Description
Make Webhook Sends data payload triggers to a configured webhook endpoint.
n8n Webhook Dispatches trigger data payloads specifically to an n8n workflow webhook endpoint.
Zapier Webhook Dispatches trigger data payloads specifically to a Zapier webhook integration endpoint.
