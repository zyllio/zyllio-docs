API SaaS

Available from the Enterprise plan

Introduction

This documentation describes the available APIs for setting up a SaaS application that lets you automate:

  • Creating applications from another application template
  • Publishing applications
  • Updating applications
  • Synchronizing applications

The URL prefix is: https://www.zyllio.one/api/account/

Endpoint documentation

Method Path Headers Body Comment Response
GET /applications x-api-key - Lists the applications linked to the API key ApplicationSummaryModel[]
POST /applications x-api-key CreateApplicationModel Creates an application from a template string
PUT /sync-applications/:id x-api-key SyncApplicationModel Synchronizes an application void
PUT /applications/:id x-api-key UpdateApplicationModel Updates an application void
POST /publications/:id x-api-key - Publishes an application void

Define the REST APIs

You need to define 2 configurations in the REST settings

  • applications - to create and update applications
  • publications - to publish applications

List applications
