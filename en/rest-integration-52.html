REST integration

Connect your app to external APIs for read/write operations with controlled flows.

Integration setup

  1. Define endpoint purpose and expected payload contracts.
  2. Store base URLs and tokens in secure configuration variables.
  3. Map API responses to your UI model in Screen editor.

Reliability practices

  • Handle timeout and HTTP errors explicitly.
  • Provide user feedback for loading and failure states.
  • Validate fallback behavior in Simulation.

Security baseline

Never expose raw secrets in client views. Route sensitive operations through protected backend endpoints when needed.
