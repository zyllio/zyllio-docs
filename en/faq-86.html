Which date format is supported by Zyllio

Zyllio natively supports the ISO format: YYYY-MM-DD HH:mm

Which date format should I use for my TimeTonic date column

The date format defined on the TimeTonic column must be YYYY-MM-DD

Why is my Navigation component no longer visible in Tablet/Desktop mode

The Navigation component requires a Header component to be present, where it is placed
