First application

Build your first Zyllio app from idea to a runnable prototype in a few guided steps.

Before you start

Make sure you have a clear app goal, one target user flow, and sample data for your first screen. Start small and iterate quickly.

  • Define one business scenario to deliver first value.
  • Prepare 3 to 5 data fields you want to display or edit.
  • Decide your first publication target: preview, test, or production.

Step-by-step

  1. Create a new application from your workspace and set project name, language, and orientation.
  2. Design your first screen in the Screen editor.
  3. Add navigation logic in Screen sequence editor.
  4. Bind your UI to data (Google Sheets, Airtable, API, or variables).
  5. Run and validate behavior in Simulation.
  6. Prepare distribution from Publications.

Validation checklist

  • Your first screen loads without missing bindings.
  • Navigation works in both nominal and empty-data cases.
  • Main actions are visible and easy to trigger on mobile.
  • You can run the app end-to-end in simulation.
