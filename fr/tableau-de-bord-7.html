Tableau de bord

Le tableau de bord constitue le point central de contrôle pour l'administration de votre espace de création d'applications. Cette page vous offre un accès rapide aux fonctionnalités essentielles, facilitant la gestion de vos applications et la navigation à travers les différents aspects de la plateforme.

Gestion des applications

La section "Applications" vous permet de créer, supprimer, cloner et éditer vos applications.

Actions disponibles

  • Créer une nouvelle application
  • Supprimer une application existante
  • Cloner une application pour gagner du temps dans le développement ou bien faire une sauvegarde
  • Éditer les paramètres et les fonctionnalités de chaque application
