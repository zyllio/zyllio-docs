Domaine personnalise

Vous avez la possibilite de definir un domaine personnalise pour que vos utilisateurs puissent acceder a votre application Zyllio via une URL personnalise, comme par exemple : www.mon-app.com.

Disponible depuis le plan Professional.

Ce guide vous explique comment configurer votre propre domaine personnalise pour votre application Zyllio.

Activer le domaine personnalise

Depuis l'interface Zyllio, puis Settings, accedez a la section Custom Domain (Domaine personnalise).

Activez l'option en basculant le bouton Enable.

Renseigner votre domaine personnalise

Dans le champ Domaine, saisissez le domaine que vous souhaitez associer a votre application.

Par exemple : www.mon-app.com.

Assurez-vous d'avoir achete le domaine et d'en avoir le controle via un fournisseur de domaine (ex: OVH, GoDaddy, etc.).

Demander la configuration DNS

Une fois que vous avez renseigne votre domaine, cliquez sur le bouton Request DNS configuration.

Une fois votre demande soumise, l'equipe Zyllio traitera votre demande dans un delai de 24 heures.

Vous recevrez un e-mail contenant les parametres DNS a saisir dans l'interface d'administration de votre fournisseur de domaine. Ces parametres peuvent inclure des enregistrements de type CNAME ou A, selon la configuration specifique.

Configurer votre DNS

Connectez-vous a l'interface d'administration de votre fournisseur de domaine.

Ajoutez les parametres DNS fournis par Zyllio dans la section dediee aux enregistrements DNS.

Type Name Value
A 216.239.38.21
AAAA 2001:4860:4802:38::15
CNAME www ghs.googlehosted.com.
CNAME <fourni par email> <fourni par email>

Propagation DNS et acces au domaine

Une fois les enregistrements DNS configures, la propagation peut prendre jusqu'a 24-48 heures.

Une fois la propagation terminee, vos utilisateurs pourront acceder a votre application via votre domaine personnalise, par exemple : www.mon-app.com.
