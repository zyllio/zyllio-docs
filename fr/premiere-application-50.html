Première application

Créez votre première application Zyllio, de l'idée au prototype exécutable, en quelques étapes guidées.

Avant de commencer

Préparez un objectif clair, un parcours utilisateur principal et un petit jeu de données pour votre premier écran.

  • Définir un scénario métier prioritaire.
  • Préparer 3 à 5 champs de données à afficher ou modifier.
  • Choisir la cible de publication: prévisualisation, test ou production.

Étapes principales

  1. Créer une nouvelle application puis définir nom, langue et orientation.
  2. Concevoir le premier écran dans l'Editeur d'ecrans.
  3. Définir la navigation dans l'Editeur de sequence d'ecrans.
  4. Lier l'interface aux données (Google Sheets, Airtable, API ou variables).
  5. Valider le comportement dans la Simulation.
  6. Préparer la distribution depuis Publications.

Checklist de validation

  • Le premier écran se charge sans liaison manquante.
  • La navigation fonctionne avec et sans données.
  • Les actions principales sont visibles et simples sur mobile.
  • Le parcours complet est validé en simulation.
