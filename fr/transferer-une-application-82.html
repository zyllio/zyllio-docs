Transférer une application

Cette page présente les différentes étapes à suivre pour transférer une application. Vous y trouverez des informations pratiques et des conseils afin de faciliter la passation, assurer la continuité du projet et garantir la satisfaction de l’ensemble des parties prenantes.

Accéder au Studio Zyllio

Accédez au Studio Zyllio Français via ce lien : https://www.zyllio.one

Connectez vous à l’aide de votre compte Google pour afficher le tableau de bord Zyllio ci-dessous

Souscrire l’abonnement Entreprise

Cliquez sur le bouton situé en haut à gauche

Sélectionnez l’abonnement Enterprise

Procédez au paiement, la navigation revient automatiquement sur votre tableau de bord Zyllio avec votre abonnement immédiatement activé

Transférer une application

Accédez à vos applications partagées depuis le menu Shared comme suit:

Cliquez sur le menu de l’application puis Clone

Félicitations, votre application est maintenant disponible dans votre compte, revenez dans le menu All pour accéder à vos applications

Lorsque vous clonez votre application, vous obtenez une copie indépendante qui vous appartient exclusivement et qui n’est accessible à aucun autre utilisateur, sauf si vous choisissez de la partager
