Tutoriels

Un parcours pratique pour maîtriser les intégrations, l'automatisation et les fonctionnalités avancées.

Parcours d'apprentissage recommandé

  1. Commencer par les bases data: Google Sheets et Airtable.
  2. Poursuivre avec logique et état: Formules et Variables.
  3. Ajouter les services externes: Integration Rest et Authentification.
  4. Renforcer l'engagement: Push Notifications, Chat et Paiements Stripe.

Sujets avancés

  • Mode deconnecte offline pour les contextes à faible connectivité.
  • Domaine Personnalise pour un déploiement web de marque.
  • Traductions pour les applications multilingues.
  • Xano, Make et Zapier pour les chaînes d'automatisation.
