Outil de conception

Cette section détaille les différents outils de conception disponibles dans notre plateforme, ainsi que l'accès aux paramètres essentiels.

Éditeur d'écran

L'éditeur d'écran est l'outil central pour la conception visuelle de chaque écran de votre application. Utilisez cet éditeur intuitif pour créer des interfaces utilisateur attrayantes et fonctionnelles.

Fonctionnalités

  • Ajout simple de composants tels que boutons, champs de texte, listes, images, etc.
  • Personnalisation de la mise en page et du style
  • Aperçu en temps réel des modifications apportées

Éditeur de séquence d'écran

L'éditeur de séquence d'écran vous permet de définir le flux de navigation de votre application en définissant l'enchaînement logique des écrans.

Fonctionnalités

  • Création de scénarios de navigation
  • Gestion des transitions entre les écrans
  • Visualisation du parcours utilisateur
  • Définition de l’écran de démarrage

Simulateur d'application mobile

Utilisez le simulateur d'application mobile pour prévisualiser l'apparence et le comportement de votre application sur différents appareils et résolutions.

Options disponibles

  • Sélection de différents modèles d'appareils mobiles
  • Test de l'interface utilisateur et des fonctionnalités

Éditeur de base de données et tables

L'éditeur de base de données vous permet de modéliser la structure de votre base de données et de définir les tables nécessaires pour stocker les données de votre application.

Fonctionnalités

  • Création et gestion de tables
  • Définition de relations entre les tables
  • Import/export de données

Accès aux paramètres

Cette section offre un accès centralisé aux paramètres essentiels de votre application.

Informations de l'application

Consultez et modifiez les informations clés de votre application, telles que le nom, l'icône, la version, etc.

Détails accessibles

  • Nom et description de l'application
  • Icône et visuels associés
  • Informations sur la version
  • Votre email de notifications

Intégrations

Gérez les intégrations avec d'autres services et plateformes pour enrichir les fonctionnalités de votre application.

Actions possibles

  • Ajout d'intégrations tierces comme Zapier, Make, Twilio, Airtable, TimeTonic, ...
  • Configuration des paramètres d'intégration

Plugins

Explorez et activez les plugins pour étendre les fonctionnalités de votre application de manière modulaire.

Fonctionnalités des plugins

  • Liste des plugins disponibles
  • Activation/désactivation des plugins
  • Configuration spécifique aux plugins
