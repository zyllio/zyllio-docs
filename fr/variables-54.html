Variables

Pilotez l'état de l'application avec les variables pour synchroniser les écrans et fiabiliser les interactions.

Stratégie de variables

  • Utiliser des variables locales pour les valeurs temporaires d'un écran.
  • Utiliser des variables partagées pour le contexte de session.
  • Réinitialiser les variables sensibles à la déconnexion ou fin de parcours.

Mise en place pratique

  1. Définir une nomenclature par scope et usage.
  2. Initialiser des valeurs par défaut sur les écrans d'entrée.
  3. Mettre à jour l'état via des actions explicites et des formules.
  4. Vérifier les transitions d'état dans l'Editeur de sequence d'ecrans.

Intégrité d'état

Documentez le cycle de vie attendu des variables pour éviter les valeurs obsolètes et les effets de bord.
