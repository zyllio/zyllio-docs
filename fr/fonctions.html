Guide des Fonctions de Zyllio Studio

Ce document fournit une liste complète de toutes les fonctions de Zyllio avec leurs descriptions (hints).

Condition

Libellé Description
Si alors Un bloc conditionnel simple qui renvoie une valeur spécifique uniquement si la condition définie est remplie.
Si alors sinon Un bloc logique conditionnel qui renvoie une valeur si la condition est vraie, et une autre si elle est fausse.

Données

Libellé Description
Compter lignes Renvoie le nombre total de lignes dans une liste ou une sélection de table spécifiée.
Compter lignes uniques Compte le nombre de valeurs distinctes et uniques dans une colonne sur les lignes d'une table.
Lignes en texte Concatène et formate les valeurs d'une colonne sur plusieurs lignes en un seul bloc de texte.
Moyenne dans lignes Calcule la valeur numérique moyenne d'une colonne spécifiée sur un ensemble de lignes de table.
Nombre maximal dans lignes Trouve la valeur numérique la plus élevée dans une colonne spécifiée sur un ensemble de lignes.
Nombre minimal dans lignes Trouve la valeur numérique la plus basse dans une colonne spécifiée sur un ensemble de lignes.
Rechercher dans lignes Filtre et recherche les lignes de table correspondant aux critères de requête spécifiés.
Somme dans lignes Calcule la somme totale de toutes les valeurs numériques d'une colonne sur un ensemble de lignes.
Texte maximal dans lignes Trouve la valeur textuelle maximale (la dernière par ordre alphabétique) dans une colonne sur les lignes.
Texte minimal dans lignes Trouve la valeur textuelle minimale (la première par ordre alphabétique) dans une colonne sur les lignes.

Date

Libellé Description
Ajouter Ajoute une quantité de temps spécifiée (jours, heures, minutes) à une date de base.
Date et heure Construit ou formate une valeur de date et heure personnalisée à partir d'entrées spécifiques.
Date et heure actuelles Renvoie la date et l'heure actuelles du système ou de l'appareil.
Durée depuis aujourd’hui Calcule la durée ou le temps écoulé entre une date spécifiée et le jour actuel.
Durée entre deux dates Calcule la durée ou la différence de temps entre deux valeurs de date spécifiées.
Soustraire Soustrait une quantité de temps spécifiée (jours, heures, minutes) d'une date de base.

Géolocalisation

Libellé Description
Distance Calcule la distance en ligne droite entre deux coordonnées géographiques spécifiées.
Distance depuis la position Calcule la distance géographique entre une coordonnée définie et la position actuelle de l'utilisateur.

Image

Libellé Description
Image Traite ou résout les URL sources d'images et les options d'affichage.

JSON

Libellé Description
Requête JSON Interroge et extrait des valeurs spécifiques d'un objet JSON à l'aide d'une expression de recherche.

Élément de liste

Libellé Description
Colonne Récupère la valeur d'une colonne spécifique à partir d'un élément de liste ou d'une ligne de base de données.
Index de l’élément de liste Renvoie l'index (commençant par 0) de la position d'un élément au sein d'une liste ou sélection.
Position de l’élément de liste Renvoie la position d'affichage (commençant par 1) d'un élément au sein de sa liste parente.

Nombre

Libellé Description
Calcul Évalue une expression mathématique combinant des nombres, des variables et des opérateurs standards.
Montant Formate une valeur numérique sous forme de devise avec le symbole et les décimales configurés.
Nombre Convertit ou formate les entrées dans un format numérique normalisé.
Nombre aléatoire Génère une valeur numérique pseudo-aléatoire dans une plage minimale et maximale spécifiée.

Paiement

Libellé Description
Afficher le panier Génère une représentation textuelle formatée du contenu du panier pour impression ou reçu.
Montant du panier Calcule le montant monétaire total de tous les articles actuellement dans le panier.
Nombre de produits du panier Renvoie le nombre total de produits uniques actuellement ajoutés au panier.
Quantite du panier Renvoie la quantité totale cumulée de tous les articles du panier.

SaaS

Libellé Description
Id de l'application Récupère l'identifiant unique de l'instance d'application en cours d'exécution.
Id de table Récupère l'identifiant unique de base de données pour un nom de table spécifié.
Id du modèle d'application Récupère l'identifiant du modèle d'application associé à l'application en cours d'exécution.
Paramètre de l'application Récupère la valeur d'un paramètre de configuration global spécifique de l'application.

Texte

Libellé Description
Capitaliser Convertit la première lettre du texte en majuscule et le reste en minuscule.
Concaténer Joint plusieurs segments de texte ou valeurs de variables en une seule chaîne continue.
Diviser Divise une chaîne de texte en un tableau de sous-chaînes en fonction d'un séparateur.
Expression régulière Vérifie si une chaîne de texte correspond à un modèle d'expression régulière spécifié.
Identifiant unique Génère une chaîne d'identifiant unique aléatoire (UUID/GUID).
Inclut Vérifie si un terme de recherche spécifié existe dans une chaîne de texte cible.
Joindre Concatène les éléments d'une liste en une seule chaîne à l'aide d'un caractère séparateur.
Longueur Renvoie le nombre de caractères d'une chaîne de texte spécifiée.
Mettre en majuscule Convertit tous les caractères du texte saisi en majuscules.
Mettre en minuscule Convertit tous les caractères du texte saisi en minuscules.
Occurences Compte le nombre de fois qu'un terme de recherche apparaît dans une chaîne de texte cible.
Remplacer Remplace les occurrences d'une chaîne de recherche spécifiée par une chaîne de remplacement.
Remplacer l'expression Remplace les sous-chaînes correspondant à un modèle d'expression régulière par une chaîne de remplacement.
Répéter le texte Répète une chaîne de texte spécifiée un nombre défini de fois.
Sous-chaîne Extrait une partie d'une chaîne de texte à partir d'un index spécifié avec une longueur définie.
Texte Convertit une valeur ou variable en sa représentation textuelle normalisée.
Valider un IBAN Vérifie si une chaîne de texte est un numéro de compte bancaire international (IBAN) valide.

Terminal

Libellé Description
Identifiant terminal Récupère l'identifiant unique du terminal ou de l'appareil de l'utilisateur.
Langue du terminal Récupère le paramètre de langue principale du terminal ou du navigateur de l'utilisateur.
Système d’exploitation Récupère le nom du système d'exploitation du terminal ou de l'appareil de l'utilisateur.
Type de terminal Récupère la catégorie ou le type d'appareil (ex. ordinateur, mobile, tablette) du terminal.

Traduction

Libellé Description
Traduire Traduit une clé ou un libellé dans la langue active de l'interface utilisateur.
