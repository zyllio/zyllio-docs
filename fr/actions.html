Documentation

Guide des Actions de Zyllio Studio

Ce document fournit une liste complète de toutes les actions de Zyllio avec leurs descriptions (hints).

Compte

Libellé Description
Créer application Crée une nouvelle instance d'application dans l'espace de travail du compte.
Modifier application Met à jour les métadonnées, paramètres et préférences de l'application.
Publier application Publie la version actuelle de l'application, la rendant opérationnelle et accessible.
Synchroniser application Synchronise l'état et les configurations de l'application dans les environnements.

Authentification

Libellé Description
Afficher le profil utilisateur Affiche le panneau de gestion du profil utilisateur pour modifier les informations personnelles.
Changer le mot de passe Déclenche une demande de modification de mot de passe pour l'utilisateur connecté.
Connexion Authentifie un utilisateur avec son identifiant et mot de passe, ouvrant une session sécurisée.
Connexion SSO Initie un flux de connexion d'authentification unique (SSO) via les fournisseurs d'identité.
Déconnexion Déconnecte l'utilisateur actuel, efface les jetons de session et redirige vers la connexion.
Déconnexion SSO Déconnecte l'utilisateur de sa session d'authentification unique (SSO).
Inscription Enregistre un nouveau compte utilisateur avec les identifiants et informations spécifiés.
Réinitialiser le mot de passe Envoie un e-mail ou un code de réinitialisation de mot de passe pour la récupération de compte.

Bluetooth

Libellé Description
Connexion Bluetooth Invite l'utilisateur à sélectionner et à se connecter à un appareil Bluetooth (BLE) à proximité.
Écriture Bluetooth Écrit des données sur une caractéristique de service d'un appareil Bluetooth connecté.
Notification Bluetooth S'abonne aux notifications ou indications d'une caractéristique d'un appareil Bluetooth connecté.

Calendrier

Libellé Description
Ajouter un événement Crée et ajoute un événement personnalisé directement dans le calendrier de l'appareil.

Contrôles

Libellé Description
Continuer si Arrête l'exécution du flux de travail si une condition spécifiée n'est pas validée.
Début boucle Démarre une boucle pour répéter des actions imbriquées sur une liste d'éléments.
Fin boucle Marque la limite de fin d'un bloc de boucle dans la séquence du flux de travail.
Point de décision Évalue une condition pendant l'exécution de l'action pour aiguiller le flux de travail.

Données

Libellé Description
Créer ligne Insère une nouvelle ligne d'enregistrement dans une table avec les valeurs configurées.
Exporter CSV Génère et déclenche le téléchargement d'un fichier CSV contenant les lignes d'une table.
Mettre à jour ligne Met à jour les valeurs de colonnes d'une ligne d'enregistrement spécifique dans une table.
Mettre à jour sélection Met à jour les valeurs de la ligne sélectionnée dans un composant de liste ou de table.
Supprimer des lignes Supprime plusieurs lignes d'enregistrement d'une table selon les filtres de requête.
Supprimer sélection Supprime la ligne actuellement sélectionnée dans un composant de liste ou de table.
Supprimer une ligne Supprime une ligne d'enregistrement spécifique d'une table correspondant à un identifiant.

Appareil

Libellé Description
Dicté vocale Démarre l'enregistrement de la voix et active la reconnaissance vocale sur l'appareil.
Obtenir position Interroge les capteurs GPS de l'appareil pour obtenir les coordonnées géographiques actuelles.

Dialogue

Libellé Description
Afficher confirmation Affiche une boîte de dialogue demandant à l'utilisateur de confirmer ou d'annuler une action.
Afficher message Affiche une boîte de message d'alerte modale personnalisée avec des boutons.
Afficher notification Affiche une bannière temporaire d'alerte de notification sur l'écran.

Messagerie

Libellé Description
Envoyer email Envoie un e-mail via Gmail avec le destinataire, l'objet et le corps spécifiés.
Envoyer message WhatsApp Envoie un message WhatsApp à l'aide de l'intégration Twilio.
Envoyer SMS Envoie un message SMS à l'aide de l'intégration de la passerelle Twilio.

Navigation

Libellé Description
Afficher popup Ouvre l'écran désigné sous la forme d'une fenêtre contextuelle (popup) modale.
Aller à l'accueil Redirige directement vers l'écran d'accueil désigné de l'application.
Aller vers écran Navigue directement vers un écran cible spécifié par son nom ou son identifiant.
Attendre Suspend l'exécution du flux de travail pendant une durée spécifiée en secondes.
Fermer popup Ferme la fenêtre contextuelle (popup) actuellement active.
Naviguer vers lien Redirige le navigateur ou l'application vers une adresse URL externe.
Rafraîchir écran Recharge les données et rafraîchit tous les composants de l'écran actif.
Retour Redirige vers l'écran précédent dans l'historique de navigation.

Hors ligne

Libellé Description
Synchro. hors ligne Déclenche la synchronisation des données locales hors ligne avec le serveur distant.

OpenAI

Libellé Description
Générer du texte Utilise les modèles OpenAI GPT pour générer des textes ou réponses à partir d'invites.
Générer image Utilise OpenAI Dall-E pour générer des images personnalisées à partir de descriptions textuelles.
Synthèse vocale Synthétise un fichier audio parlé à partir de textes avec les modèles de voix OpenAI.
Transcription audio Transcrit des enregistrements vocaux en texte brut avec le modèle OpenAI Whisper.

Paiement

Libellé Description
Ajouter produit Ajoute un produit et sa quantité associée au panier d'achat actif.
Annuler paiement Annule le processus de paiement ou de souscription d'abonnement en cours.
Confirmer paiement Confirme et valide une transaction de paiement en attente.
Créer un compte Enregistre un compte ou profil utilisateur auprès de services de paiement.
Payer Initie le processus de paiement pour les articles du panier d'achat.
Payer un abonnement Initie le processus d'abonnement à facturation récurrente pour un forfait choisi.
Portail client Redirige l'utilisateur vers son portail de facturation et de gestion de paiement.
Réinitialiser panier Vide entièrement le panier d'achat actif et réinitialise les totaux.
Retirer un produit Retire des articles ou diminue leur quantité dans le panier d'achat.

Notification push

Libellé Description
Envoyer à tous Diffuse une notification push à l'ensemble des utilisateurs de l'application.
Envoyer à un utilisateur Envoie une notification push ciblée vers un identifiant d'utilisateur spécifique.
Envoyer au canal Envoie une notification push aux utilisateurs abonnés à un canal spécifique.
S'abonner Abonne le terminal aux alertes de notifications push de l'application.

Code QR

Libellé Description
Scanner QR Code Ouvre l'appareil photo du terminal pour scanner et décoder un code QR.

REST

Libellé Description
Delete Envoie une requête HTTP DELETE à un point de terminaison d'API REST spécifié.
Get Envoie une requête HTTP GET pour récupérer des ressources depuis une API REST.
Patch Envoie une requête HTTP PATCH pour modifier partiellement une ressource via une API REST.
Post Envoie une requête HTTP POST pour soumettre des données à une API REST.
Put Envoie une requête HTTP PUT pour remplacer ou écrire une ressource via une API REST.

Thème

Libellé Description
Agrandir texte Augmente la taille de police globale du thème (zoom avant du texte).
Réduire texte Diminue la taille de police globale du thème (zoom arrière du texte).

Variable

Libellé Description
Charger les variables Charge et initialise des variables dans l'état de session de l'application.
Décrémenter Diminue de 1 la valeur numérique d'une variable spécifiée.
Définir la sélection Assigne une variable de sélection pour pointer vers un élément de ligne cible.
Définir la variable si Assigne une valeur à une variable uniquement si une condition spécifiée est validée.
Définir une variable Assigne une valeur statique ou dynamique spécifiée à une variable cible.
Effacer la sélection Réinitialise le pointeur ou la variable de sélection de ligne à vide.
Incrémenter Augmente de 1 la valeur numérique d'une variable spécifiée.
Réinitialiser les variables Réinitialise toutes les variables de session ou une variable cible à sa valeur par défaut.

Webhook

Libellé Description
Webhook Make Envoie des données de déclenchement à un point de terminaison webhook configuré.
Webhook n8n Envoie des données de déclenchement à un webhook de flux de travail n8n.
Webhook Zapier Envoie des données de déclenchement à un webhook d'intégration Zapier.
