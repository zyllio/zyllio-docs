Simulation

Le simulateur permet de tester des applications mobiles dans un environnement simulé, offrant ainsi la possibilité de visualiser et d'interagir avec des applications sur différents modèles de téléphones mobiles. Il prend en charge à la fois les plateformes Android et iOS

Ce simulateur d'applications mobiles est un outil efficace pour tester les fonctionnalités de votre application sur différents appareils et systèmes d'exploitation. Il permet également de simplifier le processus de test en proposant une authentification rapide via la connexion automatique. Utilisez-le pour vérifier la performance et l'expérience utilisateur de vos applications dans des conditions proches de la réalité

Choix du modèle d'appareil :

Le simulateur offre un large éventail de modèles d'appareils parmi lesquels vous pouvez choisir pour tester votre application. Vous pouvez sélectionner des téléphones Android et iOS populaires comme des Samsung Galaxy, iPhones

Pour changer de modèle d'appareil, utilisez le menu déroulant sur la gauche de l'interface. Cela permet d’adapter l’affichage de l’application selon les spécifications de l'appareil choisi, y compris la résolution d’écran et les particularités du système d’exploitation

Authentification automatique

Le simulateur dispose d'une fonctionnalité d'authentification automatique, permettant de se connecter à l'application sans avoir à saisir manuellement l'email et le mot de passe.

Pour utiliser cette fonctionnalité, vous devez sélectionner un utilisateur à authentifier à partir du menu dédié. Cette option permet de simuler la connexion rapide à une application sans entrer d’informations de connexion à chaque fois

Pour que l'authentification automatique fonctionne correctement, la table d'authentification contenant trois colonnes obligatoires :

  • Email : l'email de l'utilisateur
  • Name : le nom de l'utilisateur
  • Password : le mot de passe de l'utilisateur

Si ces colonnes ne sont pas présentes, la fonctionnalité d'authentification automatique ne pourra pas être utilisée. Il est donc essentiel de vérifier la structure de la table avant d’activer cette option

Comment utiliser le simulateur

Sélection de l'appareil

Accédez au menu déroulant en haut à gauche pour choisir un modèle de téléphone. Une fois sélectionné, le simulateur ajustera automatiquement l'affichage pour correspondre à l'apparence de cet appareil

Authentification automatique

Pour activer l'authentification automatique, sélectionnez l'option correspondante dans le menu "Act as" (Agir en tant que) et choisissez un utilisateur prédéfini. Cela simulera l’expérience d’un utilisateur se connectant directement à l'application sans saisir ses informations manuellement
